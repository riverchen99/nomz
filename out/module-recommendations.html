(inner) propsCheck(infoArray, props, type) → {Boolean}

Auxiliary function to return whether a menuItem is included in the filtered results based on whether its props (dietary tags) are compatible with the user's filters. If checking for preferences, a menu item should have all or * most of props identified in the user preferences. If checking for restrictions, a menu item must not have any props that the user restricted on order to be included.
Parameters:
Name Type Description
infoArray Array.<string> Contains a list of either user restrictions or preferences to check
props MenuItem.props Desc
type string 'preferences' or 'restrictions'. Used to relate any matches to type of filtering.
